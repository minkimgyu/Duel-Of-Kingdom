## 프로젝트 소개
Unity를 사용하여 개발한 멀티플레이 1vs1 3D 액션 게임

## 개발 기간
23. 12 ~ 24. 03

## 인원
3명 (서버 프로그래머 1명, 클라이언트 프로그래머 2명)

## 개발 환경
* Unity (C#)

## 기능 설명

* ### 상속을 활용하여 Unit, Tower 구현

* ### FSM과 Behavior Tree를 활용하여 AI 시스템 구축

* ### A* 알고리즘을 활용한 길 찾기 알고리즘 구현 및 적용
